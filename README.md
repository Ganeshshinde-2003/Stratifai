# Stratifai — AI Marketing Intelligence Engine

Generate a complete marketing strategy (ads, emails, landing pages, SEO, competitor insights) from a product URL in ~60 seconds.

## 🚀 Key Highlights

- Multi-step LLM pipeline (not a single prompt)
- Handles real-world constraints: structured output, validation, retry logic
- Production-ready full-stack system (FastAPI + Next.js)
